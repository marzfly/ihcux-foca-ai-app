## Visão Geral
Integrantes: Arthur Cantarutti, Bernardo Rocha, Guilherme Augusto, Gabrielly Cunha, Mariana Duarte, Thais Augusta  

O sistema foi desenvolvido para ajudar usuários a manterem o foco em atividades através de sessões colaborativas, reduzindo distrações e ansiedade causadas por excesso de estímulos digitais.

---

# Conceito de Design

A interface foi projetada para transmitir simplicidade e conforto visual.

Principais decisões:

* Layout minimalista.
* Poucos elementos por tela.
* Cores suaves e neutras.
* Feedbacks discretos.
* Redução de notificações e distrações durante a sessão.

O objetivo é manter o usuário concentrado sem gerar sobrecarga visual ou pressão.

---

# Fluxo de Usuário

1. O usuário acessa a plataforma.
2. Escolhe entrar ou criar uma sessão.
3. Configura duração e preferências da sessão.
4. Inicia o modo de foco.
5. Acompanha o temporizador e progresso.
6. Finaliza a sessão e registra resultados.

---

# Escolha de Componentes

## Slider para tempo da sessão

Foi utilizado para facilitar a configuração rápida e reduzir erros de digitação.

## Botões grandes

Melhoram acessibilidade e evitam cliques incorretos.

## Cards para sessões

Organizam informações de forma visual e intuitiva.

## Toggle para opções rápidas

Permite ativar recursos sem abrir menus complexos.

---

# Objetivo do Projeto

Criar um ambiente digital simples e confortável que incentive foco, produtividade e bem-estar.
